##Instructions

##chatwo - an Acoustical chat app
usage: chatwo.py [-h] -p PORT -s SERVER -n NAME

optional arguments:
  -h, --help            show this help message and exit
  -p PORT, --port PORT  server port
  -s SERVER, --server SERVER
                        server address
  -n NAME, --name NAME  your public handle
